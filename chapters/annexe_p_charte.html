Charte de Qualité & Règles d'Or

Le socle de valeurs non négociables. Les 4 Commandements et la matrice de conformité.

Les 4 Commandements (P.1)

1. No Broken Windows

Aucun code n'est mergé sur 'main' s'il contient des warnings de linter ou des TODOs non résolus.

2. Don't Trust, Verify

Chaque ligne de code générée par IA doit être auditée humainement. L'IA propose, l'Humain dispose.

3. Ships or Nothing

Un projet non déployé n'existe pas. Le code local ne compte pas. Production Only.

4. Leave No One Behind

Le savoir ne vaut que s'il est partagé. Le peer-learning est une obligation, pas une option.

Matrice de Conformité (P.2)

Infraction Exemple Sanction
Plagiat Copie repo externe sans crédit EXCLUSION
Négligence Sécu Commit Private Key Blâme + Reset
Ghosting Absence > 48h sans justif Avertissement
