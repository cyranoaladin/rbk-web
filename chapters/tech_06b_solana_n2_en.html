LEVEL 2: SMART CONTRACT DEVELOPMENT

This level covers the full lifecycle of decentralized application (dApp) development. The approach combines Anchor framework for productivity and Native development study for deep technical understanding.

1. Anchor Framework

Weeks 5-6
  • Anchor Architecture: Macros, Context, and automatic account validation.
  • Data Management: Practical implementation of PDAs and account relationships.
  • CPI (Cross Program Invocation): Interaction with other programs (e.g., SPL Token).
  • Client Integration: IDL generation and Frontend connection (Web3.js).

2. Native Development

Weeks 7-8
  • Program Entrypoint: Low-level instruction processing (`process_instruction`).
  • Serialization: Manual data handling with Borsh.
  • Explicit Security: Manual verification of signers and account owners.

LEVEL 2 PROJECT: "Full Stack dApp"

Creation of a complete decentralized application (e.g., Crowdfunding Protocol or simple DAO).
Deliverables: Optimized Smart Contract (Anchor), unit test suite, and functional user interface connected to wallets.
