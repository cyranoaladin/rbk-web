TRACK A

Solana (Rust) : Parcours Builder

Ce track forme des builders Solana capables de concevoir, coder, tester et auditer des programmes on-chain. La progression suit une logique "senior-friendly" : d’abord le modèle mental, puis l'efficacité (Anchor), la profondeur (Native), la sécurité, et enfin l'architecture (AMM, Tokens).

1) Progression Officielle

01
Foundations

Théorie Solana & Mental Model

Accounts, transactions, PDA, CPI, SPL token mental model.

N1
Rust System + Tools

Bas niveau & Outillage

Rust basique ciblé Solana, sérialisation, logs, lecture d’exécution.

N2
Anchor Framework

Counter → Patterns

Premier programme complet, contraintes, PDA, CPI de base, tests TypeScript.

N3
Anchor → Native

Capstone & Validation

Capstone Anchor complexe, puis reproduction en natif (validation manuelle & invoke_signed).

05
Security Labs

Vulnerability Hunting

Trouver, expliquer et corriger des vulnérabilités (Anchor + Native) avec preuves par tests.

06
Tokens & CPI

Standards Avancés

SPL Token, Token-2022 Extensions (à bon escient), et CPIs propres.

07
Build an AMM

Protocol Engineering

AMM minimal → features avancées (fees, LP, slippage, compute budget en option).

08
Validator/Infra

Ouverture Infrastructure

Culture réseau, exécution, RPC, performance et bonnes pratiques.

2) Objectifs Finaux

Expliquer le modèle d’accounts et la mécanique transaction/instruction (Mental Model).
Écrire des programmes Anchor propres : contraintes, PDA, CPI, tests, events.
Lire et raisonner sur du Native Solana (validation manuelle, invoke_signed).
Identifier et corriger les vulnérabilités courantes (avec preuves).
Implémenter des flows tokens et construire un AMM minimal.

3) Livrables Attendus

Repo Counter Anchor basic + tests
Repo Vault/Escrow PDA + CPI + Errors
Capstone Anchor Protocol complet
Version Native Instruction decoding
Security Labs Corrections + PoCs
Tokens & AMM Invariants & Tests

4) Scope & Limites

  • • Accounts / Transactions (Theory)
  • • Anchor Mastery (Counter → Capstone)

  • • AMM Minimal & Token Flows
