N3

Niveau 3 — Architecte Studio & Validator Ops

20 semaines pour maîtriser performance, sécurité formelle, interopérabilité et opérations de studio Web3 (protocoles et validateurs).

Phases Alpha / Beta / Gamma

Alpha (S29-S38) — Performance et observabilité : load testing, parallélisme, compute budget.

Beta (S39-S44) — Sécurité formelle, théorie des jeux, red/blue team.

Gamma (S45-S48) — Studio de production : gouvernance, compliance, simulation de lancement.

Modules approfondis

Performance & architecture

  • Shard-then-aggregate, WAL, vues matérialisées.
  • Load tests 100k TPS, chaos, recovery benchmarks.
  • Batching atomique, ALT et compute budget précis.

Sécurité formelle

  • Invariants CTL/LTL, fairness, proofs de non-déniabilité.
  • Red/Blue team 72h avec jury.
  • Audit offensif : hooks, CPI, oracle, RWA.

Interop & cross-chain

  • Wormhole, LayerZero, state proofs, architectures hybrides.
  • Pricing unifié, settlement atomique, compression d’état.

Gouvernance & RWA

  • Compliance by design (KYC/AML, geo-lock).
  • Timelocks, kill switch, reporting.
  • Tokenisation RWA : identity registry + Transfer Hook.

Launch & ops industrielles

  • Canary releases, blue-green upgrades, post-mortems.
  • Protocol launch simulation (pre / day / post).

Validator & infra

Validator Ops
  • Bare-metal, clients Agave/Jito/Firedancer, monitoring TIG.
  • Recovery playbooks, alerting, drills incidents.
Data & Storage
  • Compression d’état, indexeurs SubQuery/Sqd.
  • RPC hybrides, QoS, multi providers.
  • Stockage immuable (Arweave/Shadow Drive) pour metadata.

CI/CD & launch readiness

  • GitHub Actions : lint, tests, build Anchor, génération de proposal multisig.
  • Builds vérifiables (hash binaire aligné code source).
  • Testing suite perf + sécurité, analytics protocole.

Certification & portfolio

  • Portfolio obligatoire : 3 architectures, 1 audit formel, 1 contribution open source, 1 whitepaper, 1 présentation.
  • Certification par un board mixte (mentors, expert externe, représentant communauté).
  • Badges tiered (Junior, Senior, Principal) avec utilités (mentorat, vote, access gating).
