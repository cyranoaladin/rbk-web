Syllabus

Parcours Technique (Web2 → Web3)

Progression en trois phases : fondamentaux système, socle Web2 produit, puis spécialisation Web3 (Solana ou EVM).

Vue d'ensemble

Phase 1

Fundamentals

Rust, OS, Git, crypto, réseau.

Phase 2

Web2 Produit

API, front-end, DevOps, observabilité.

Phase 3

Web3 Tracks

Track A Solana (Rust/Anchor) ou Track B EVM (Solidity/Foundry).

Phase 1 — Fundamentals

Semaine Focus Livrable Objectif d'apprentissage
S1 OS, shells, Git internals Clone ls -la en Rust Comprendre I/O bas niveau, permissions.
S2 Memory safety (Rust) Allocateur custom Ownership, borrowing, profils d'allocation.
S3 Concurrence HTTP server multi-thread Gestion des threads, contention, observabilité.
S4 Cryptographie CLI wallet (Ed25519) Manipuler clés, signatures, formats d'adresses.

Phase 2 — Socle Web2

Backend & APIs

  • REST + OpenAPI, rate limiting, auth stateless.
  • PostgreSQL + Prisma, transactions, indexation.
  • Threat modeling API, tests négatifs, observabilité (Grafana).

Frontend & Tooling

  • React/TypeScript, accessibilité, state management léger.
  • CI/CD GitHub Actions, qualité (lint, tests), prévisualisations.
  • Monitoring front (Sentry), budgets de performance.
