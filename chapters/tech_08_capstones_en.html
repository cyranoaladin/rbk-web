The 3 signature projects

Capstone 1

Wallet & Reliability Pack

Focus: Frontend / UX / RPC Failover

User stories
  • Precise feedback (signing/confirming).
  • Auto retry (backoff).
  • Failure journal.
Tech specs
  • Next.js + TanStack Query.
  • Strict state machine.
  • RPC failover (primary + fallback).
Security checklist
  • No private keys stored.
  • Sanitized inputs.
  • Slippage handling.
Capstone 2

Tokenization & Admin Control

Focus: RBAC / Business logic / DeFi

User stories
  • SuperAdmin: appoint/revoke operator.
  • Operator: freeze suspicious account.
  • Auditor: on-chain log of admin actions.
Tech specs
  • Solidity (AccessControl) / Anchor (PDA).
  • Invariants supply = sum balances.
  • Events for every mutation.
Security checklist
  • Two-step transfer (admin).
  • Checks-Effects-Interactions.
  • Strict access control.
Capstone 3

Digital Assets & Utility

Focus: Performance / NFT / Gaming

User stories
  • NFT-based gating.
  • Daily rewards.
  • Optimized mint.
Tech specs
  • Merkle tree or signature verifier.
  • Compression (cNFT) or ERC721A.
Security checklist
  • Bot protection.
  • Anti-replay.
  • Metadata IPFS/Arweave.
