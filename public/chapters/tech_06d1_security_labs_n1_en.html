Security Labs — N1 (Foundations)

Foundations drills for Solana builders: account ownership, signer checks, seeds validation and deny-by-default patterns.

Objectives

  • Reject wrong owner/signer early with clear constraints.
  • Validate PDA seeds and store bump safely.
  • Document a minimal threat model per lab.

Prerequisites

  • Anchor basics (accounts, ctx, constraints).
  • TS tests capable of failing on signer/owner issues.
  • Understanding of PDAs and authorities.

Deliverable

3 failing tests → patches → 3 green regressions. Each bug has Bug / Impact / Fix / Proof.
