CAPSTONE PROJECTS

Competency validation is not multiple choice, but Proof of Work. Each level concludes with a tangible and auditable deliverable.

Level 1

The Rust CLI Explorer

Development of a pure Rust command-line tool (CLI), framework-free. The tool must connect to the Solana RPC, manually parse binary data from specific accounts, and display a structured state.

Rust Ownership Borsh Deserialization RPC Integration
Expected deliverables
  • Rust source code (Crates.io standard)
  • Error Handling (Result/Option)
  • Zero `unwrap()` in production
  • Technical documentation (Cargo doc)
Level 2

Full Stack DeFi dApp

Design and deployment of a complete decentralized protocol (e.g., Crowdfunding with Escrow, or Voting DAO). The project must include the Smart Contract (Anchor), unit tests, and a connected React user interface.

Anchor Framework CPI (Composability) PDA Management
Expected deliverables
  • Deployed Program (Devnet)
  • TS Tests (> 80% coverage)
  • Functional Frontend (Wallet Adapter)
  • Public IDL
Level 3

Technical Audit Report

Simulation of a "Red Teaming" mission. The student must analyze an existing (or provided) protocol, identify potential attack vectors, and produce a professional report.

Exploit Writing Vulnerability Assessment Gas Optimization
Expected deliverables
  • Industry Standard PDF Report
  • Classification (High/Med/Low)
  • PoC (Proof of Concept) of exploits
  • Fix Recommendations
