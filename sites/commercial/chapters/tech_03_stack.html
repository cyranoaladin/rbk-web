Stack de Développement

Local Env

  • OS Linux (Ubuntu) / macOS
  • Terminal Alacritty + Tmux + Starship
  • IDE VSCode (profil dédié Web3)

Chain Stack

  • Solana Rust 1.75+, Anchor 0.29+
  • EVM Foundry (Forge, Cast, Anvil)
  • Indexing The Graph / Helius
