Change Control Protocol

Gouvernance — Protocole de modification des paramètres contractuels et sections sensibles.

Protocole Obligatoire (Release Gate)

STEP 1

Update SSOT

Modifier uniquement tex/params.tex via Macros.

STEP 2

Preuve Locale

Exécuter make release-check sans erreur.

STEP 3

Preuve CI

Workflow GitHub vert + Artefact PDF généré.

STEP 4

Sign-off

Enregistrement dans la table de contrôle.

Définition des Release Gates

GO

Build OK ; Scripts PASSED ; CI Verte ; Artefact PDF conforme.

WARN

Tout OK, mais warnings LaTeX non-bloquants (ex: overfull hbox).

NO-GO

Build KO ; Script Failed ; Hardcode détecté ; Incohérence Brut/Net.
