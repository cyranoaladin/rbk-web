Charte de Qualité & Règles d'Or

Code of Conduct — Les valeurs non-négociables de l'Ingénieur RBK. "Tout manquement entraîne une exclusion."

P.1 Les 4 Commandements de l'Ingénieur RBK

01

No Broken Windows

Aucun code n'est mergé sur main s'il contient des warnings de linter ou des TODOs non résolus.
"Zero Tolerance for sloppy code."

02

Don't Trust, Verify

Chaque ligne de code générée par IA doit être auditée. L'étudiant est responsable de 100% de son output.

03

Ships or Nothing

Un projet non déployé n'existe pas. Le code local ne compte pas. La prod est la seule vérité.

04

Leave No One Behind

Le savoir ne vaut que s'il est partagé. Aider ses pairs est un devoir, pas une option.

P.2 Matrice de Conformité (Sanctions)

Infraction Exemple Concret Sanction
Plagiat Copie repo externe sans crédit / Triche IA flagrante. Exclusion Immédiate
Négligence Sécurité Commit de Private Key ou Secrets .env sur GitHub. Blâme + Reset Projet
