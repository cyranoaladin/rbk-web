Le Cockpit de l'Architecte

L'environnement de haute performance. Stack outillage obligatoire et routine de productivité "Deep Work".

Stack Outillage Minimal (F.1)

Outil Catégorie Output Attendu
Obsidian / Notion Knowledge Base Wiki du projet, Notes de recherche, Docs.
Excalidraw Diagramming Schémas d'architecture C4, Flux de données.
Linear / Jira Task Management Tickets spécifiés (User Stories) et trackés.
Cursor / VSCode IDE (Intelligent) Code propre, Linter actif, Copilot configuré.

Journée Type : Productivité (F.2)

09h - 12h : Deep Work

Focus

Coding pur. Feature complexe ou Refactoring. Règle d'or : Pas de notifications, pas de Slack/Discord.

13h - 14h : Code Review

Quality

Revue des Pull Requests (PRs) des collègues. Lecture critique, suggestions constructives.

14h - 16h : Ops & Docs

Maintenance

Écriture de tests, mise à jour de la documentation (ADR, Readme), correctifs mineurs (Hotfixes).

16h - 17h : Sync

Agile

Daily Standup, Synchro avec l'Architecte ou le Tech Lead. Déblocage des points durs.
