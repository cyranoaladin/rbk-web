La Bible Technique — Standards de Repositoire, Grilles d'Évaluation, Contrat de Livrable et Syllabus Détaillé (48 semaines).

B.1 Gabarit de Dépôt Obligatoire

repo-root/
├── README.md           # Présentation, Quickstart
├── RUNBOOK.md          # How-to run/test/debug
├── PROOFS.md           # Ledger hebdo des preuves
├── THREAT_MODEL.md     # Analyse des risques & mitigations
├── CHANGELOG.md        # Historique des versions
├── LICENSE             # MIT / Apache 2.0
├── .gitignore
├── adr/                # Architecture Decision Records
│   ├── 0001-template.md
│   └── index.md
├── docs/               # Documentation additionnelle
│   ├── architecture/
│   ├── security/
│   └── ops/
├── src/                # Code Source
├── tests/              # Tests (Unit, Integ, E2E)
├── scripts/            # Scripts d'automatisation (OBLIGATOIRES)
│   ├── setup.sh        # Installe deps
│   ├── lint.sh         # Format code
│   ├── test.sh         # Run all tests (Exit != 0 if fail)
│   ├── build.sh        # Compile artifacts
│   ├── dev.sh          # Start local env
│   ├── smoke_test.sh   # Healthcheck
│   └── deploy_devnet.sh
└── infra/
    └── setup.sh

KO-Structure (Éliminatoire)

L'absence de PROOFS.md, RUNBOOK.md, THREAT_MODEL.md ou du dossier adr/ est un échec automatique du livrable. Aucune évaluation technique n'est réalisée.

KO-Repro (Éliminatoire)

Si la séquence ./scripts/setup.sh && ./scripts/test.sh échoue sur la machine de l'évaluateur (exit code != 0), le projet est rejeté (NO-GO).

Standards de Scripts

  • Exécutables depuis la racine.
  • Non-interactifs (pas de prompt bloquant).
  • Support de l'argument --help.
  • Codes de sortie standards (0 = Succès).

B.6-7 Contrat & Syllabus Opérationnel

Contrat de Livrable Unique (Extraits)

  • Structure : PROOFS.md ordonné (PR, Tag, CI, Démo...).
  • Repro : Séquence setup && test infaillible.
  • Tests : 1 cas nominal + 1 cas d'erreur / feature.
  • Démo : Vidéo 30-90s User Story + Erreur.
  • Sécurité : Threat Model à jour + Checklist.
  • Engagement : Respect des standards dès S01.
