Partie 1

E.2 Classification des Risques (Severities)

Niveau Impact Exemples Types
CRITICAL Perte de fonds directe, Gel définitif. Reentrancy, Owner Key Compromise.
HIGH Dégradation sévère, Perte partielle. DoS, Price Oracle Manipulation.
MEDIUM Grief mineur, Coût Gas élevé. Griefing Attack, Unbounded Loop.
LOW / INFO Bonnes pratiques, Lisibilité. Typo, Dead Code, Missing Events.

E.3 Gabarit "Finding" (Fiche Type)

ID-01 : Unchecked External Call (H-01) SEV: HIGH
Fichier
vault.rs
Description
L'appel CPI vers le programme Token ne vérifie pas le code retour.
Impact
Un attaquant peut forcer l'échec silencieux du transfert et créditer son solde interne.
Fix
Utiliser anchor_lang::solana_program::program::invoke_signed et gérer le Result (Ok/Err).
