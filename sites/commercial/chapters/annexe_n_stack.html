Outillage & Stack Technique

Cyborg-Ready Environment — Toolchain Mandataire, Matrice d'Usage IA et Pipeline CI/CD Standardisé.

N.1 Stack de Développement

1. Environnement Local (L'établi)

  • Système d'Exploitation :
    Linux (Ubuntu/Pop!_OS) ou macOS. Windows via WSL2 uniquement.
  • Terminal :
    Alacritty + Tmux + Starship.
  • IDE :
    VSCode (Profile RBK : Rust, Solidity, GitLens, Copilot).

2. Chain Stack (Le Moteur)

  • S
    Solana :
    Rust 1.75+, Anchor 0.29+, Solana CLI 1.18+.
  • E
    EVM (Ethereum) :
    Foundry (Forge, Cast, Anvil).
  • I
    Indexing :
    The Graph (EVM), Helius/Shyft (Solana).

N.2 Politique & Matrice Outils IA

Usage Outil Validé Politique d'Usage Strict
Coding Assistant GitHub Copilot Autorisé pour boilerplate/tests. Interdit pour logique algo critique.
Docs Search Perplexity / Phind Recommandé pour la recherche contextuelle et debugging.
Code Review CodeRabbit Pré-analyse automatique des PRs avant review humaine obligatoire.
Diagrams Mermaid.js "Diagrams as Code" obligatoire pour toute documentation.

N.3 Infrastructure CI/CD (Github Actions)

Tout repo étudiant doit inclure un workflow standardisé .github/workflows/ci.yml :

cargo audit
cargo test
cargo fmt
cargo clippy
