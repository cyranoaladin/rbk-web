W

Glossaire & Jargon

Référentiel des termes techniques, acronymes et notions de gouvernance.

AML

Anti-Money Laundering. Lutte contre le blanchiment (surveillance flux, sanctions).

API

Application Programming Interface. Interface exposant des services consommables.

Architecte Web3

Ingénieur concevant des systèmes complets (Smart Contracts + Frontend + Indexing) sécurisés.

Bridge

Mécanisme de transfert d'actifs entre blockchains distinctes.

CI/CD

Continuous Integration/Delivery. Chaîne automatisant tests, builds et déploiements.

CLI

Command Line Interface. Interface terminal pour exécuter des scripts.

CPPS

Contrat de Partenariat Pédagogique. Cadre juridique du remboursement conditionnel.

CRM

Customer Relationship Management. Outil de gestion des leads et admissions.

DAO

Decentralized Autonomous Organization. Gouvernance par vote de détenteurs de tokens.

DeFi

Decentralized Finance. Services financiers sans intermédiaires via smart contracts.

DoD

Definition of Done. Critères minimaux pour valider un livrable.

DualTrack

Organisation en deux parcours (EVM et Solana) avec tronc commun.

ERC

Ethereum Request for Comments. Standards de l'écosystème EVM (ex: ERC-20).

EVM

Ethereum Virtual Machine. Moteur d'exécution des smart contracts Ethereum.

Fonds de Garantie

Compte séquestré couvrant les défauts ISA, gouverné avec Nexus.

Funnel

Parcours d'acquisition (Lead -> Admis) avec mesures de conversion.

Gate GO/NO-GO

Point de décision bloquant validant le passage au niveau supérieur.

IA

Intelligence Artificielle. Modèles (LLM) pour augmenter la productivité.

IP

Intellectual Property. Droits sur les créations (code, contenus).

ISA

Income Share Agreement. Dispositif d'Excellence : financement différé réservé aux lauréats (Top 2).

KO

Knock-Out. Statut d'échec critique nécessitant correction immédiate.

KPI

Key Performance Indicator. Métrique de mesure d'objectifs.

KYC

Know Your Customer. Vérification d'identité pour services réglementés.

MoSCoW

Must, Should, Could, Won't. Méthode de priorisation des tâches.

Multisig

Portefeuille nécessitant plusieurs signatures pour valider une transaction.

MVP

Minimum Viable Product. Version minimale pour valider une hypothèse.

NDA

Non-Disclosure Agreement. Accord de confidentialité.

NPS

Net Promoter Score. Indicateur de satisfaction et recommandation.

On-chain

Donnée ou logique stockée et exécutée directement sur la blockchain.

OpenAPI

Standard de description des APIs HTTP.

Ops

Pratiques d'exploitation (Monitoring, Incidents, Runbooks).

RACI

Matrice de rôles : Responsible, Accountable, Consulted, Informed.

RBAC

Role-Based Access Control. Limitation des privilèges par rôle.

RGPD

Règlement Général sur la Protection des Données.

ROI

Return On Investment. Rapport gain/coût.

RPC

Remote Procedure Call. Point d'accès pour interagir avec la blockchain.

Runbook

Guide opérationnel pour exécuter une tâche ou gérer un incident.

SaaS

Software as a Service. Logiciel par abonnement hébergé.

SBT

Soulbound Token. Jeton non transférable prouvant une compétence.

SecLead

Référent sécurité pilotant les risques et contrôles.

Senior-by-Design

Approche visant à inculquer les réflexes seniors dès le début.

SLA

Service Level Agreement. Engagement contractuel de qualité.

Smart Contract

Programme blockchain exécutant des règles immuables.

Sprint 0

Phase de 90j pour sécuriser les fondations avant lancement.

SSOT

Single Source of Truth. Donnée de référence unique.

Staking

Blocage de tokens pour sécuriser le réseau contre récompense.

TechLead

Référent technique coordonnant l'architecture et la qualité.

Threat Model

Analyse structurée des menaces et vecteurs d'attaque.

TND

Dinar Tunisien. Devise de référence du programme.

Tokenomics

Économie du token (émission, utilité, distribution).

TVL

Total Value Locked. Valeur totale des actifs dans un protocole.

Web3

Internet décentralisé basé sur la propriété numérique.

Whitelist

Liste d'adresses autorisées à interagir avec un contrat.

Anchor

Framework Solana simplifiant la sécurité et la sérialisation.

IDL

Interface Description Language. JSON décrivant l'API du programme.

Pubkey

Public Key. Adresse publique d'un compte sur la blockchain.

Reclaim

Protocole zkTLS pour prouver des données Web2 on-chain.

SPL-Token

Solana Program Library. Standard de tokens fongibles/NFTs.

Transfer Hook

Extension Token-2022 activant une logique custom à chaque transfert.

ZK Proof

Zero-Knowledge. Preuve de validité sans révéler la donnée.

CPI

Cross-Program Invocation. Un programme appelant un autre programme.

PDA

Program Derived Address. Compte sans clé privée, contrôlé par le programme.

Rent

Frais de stockage sur Solana. Remboursables si le compte est fermé.

Seeds

Graines utilisées pour dériver une adresse PDA de manière déterministe.

Sysvar

Variable Système. Compte spécial (Clock, Rent) donnant des données réseau.

Sysvar

Variable Système. Compte spécial (Clock, Rent) donnant des données réseau.

ALT

Address Lookup Table. Optimisation pour inclure plus de comptes dans une tx.

Borsh

Binary Object Representation Serializer for Hashing. Standard de sérialisation Solana.

Compute Budget

Limite de ressources (CU) allouée à une transaction pour son exécution.

Epoch

Période (~2 jours) pour laquelle le schedule des leaders est défini.

Lamport

Plus petite unité de SOL (1 SOL = 10^9 Lamports).

Mainnet Beta

Réseau principal de Solana (Production) où la valeur réelle est échangée.

Rent Exempt

État d'un compte avec solde suffisant pour ne pas payer de loyer (2 ans de stockage).

Skill Mirror

Méthodologie d'évaluation par les pairs basée sur la preuve de compétence.

Slot

Fenêtre de temps (~400ms) pour qu'un leader produise un bloc.

Validator

Nœud participant au consensus et votant sur la validité des blocs.
